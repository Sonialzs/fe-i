## 59. 使用哪个构造函数可以成功继承`Dog`类?

```js:index
class Dog {
	constructor(name) {
		this.name = name;
	}
}

class Labrador extends Dog {
	// 1
	constructor(name, size) {
		this.size = size;
	}
	// 2
	constructor(name, size) {
		super(name);
		this.size = size;
	}
	// 3
	constructor(size) {
		super(name);
		this.size = size;
	}
	// 4
	constructor(name, size) {
		this.name = name;
		this.size = size;
	}
}
```
