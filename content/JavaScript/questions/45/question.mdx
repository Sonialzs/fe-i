## 45. 以下代码会输出什么

```js:index.js
function getInfo(member, year) {
	member.name = 'Lydia';
	year = '1998';
}

const person = { name: 'Sarah' };
const birthYear = '1997';

getInfo(person, birthYear);

console.log(person, birthYear);
```
