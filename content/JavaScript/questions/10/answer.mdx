在执行到`const lydia = new Person('Lydia', 'Hallie');`时，由于通过 `new` 调用了 `Person` 方法，会创建一个对象并返回。

所以此时 `lydia` 为`{firstName: "Lydia", lastName: "Hallie"}`。

```js
const sarah = Person('Sarah', 'Smith');
```

这行代码将 Person 通过一般函数来调用，此时 this 指向全局对象，也就是 window（严格模式下是 undefined）。

Person 作为一个普通函数，默认返回值是 undefined，所以此时`sarah`的值为 `undefind`。

所以结果是：

```js
"{firstName: "Lydia", lastName: "Hallie"}"
undefined
```
