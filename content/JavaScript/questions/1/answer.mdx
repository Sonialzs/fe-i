这个问题涉及到变量提升。
`var name`提升之后相当于以下代码。

```js run
function sayHi() {
	var name;
	console.log(name);
	console.log(age);
	name = 'cuvii';
	let age = 21;
}
```

第一个 `console.log` 处，由于 `name `初始化为`undefined`，所以先输出了`undefined`。
第二个 `console.log` 处，由于 `age` 是通过 `let` 定义的，此时会发生 TDZ，报出`Cannot access "age" before initialization`。

所以结果为

```
undefined
Cannot access "age" before initialization
```
