```js
+true;
```

对于`+`操作符，有两种情况：

-   如果参与运算的两个值中有一个是字符串，会尝试将另外一个值也转换为字符串，然后进行字符串拼接。例如`'a' + 1`的结果是`"a1"`。
-   剩下情况都会将值转换为数值，然后进行相加运算。

```js run
!'Lynda';
```

将 `true` 转换为数字会得到 1，所以`+true`的结果为`1`。

`!`操作符用于取反，会将参与运算的值转换为布尔值。将字符串转换为布尔值时，只有`""`空字符串为 `false`，其他都为 `true`，所以实际上是`!true`，结果为 `false`。

所以，这道题的结果是

```js
1;
false;
```
