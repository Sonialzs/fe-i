## 46. 以下代码会输出什么

```js:index.js
function greeting() {
	throw 'Hello world!';
}

function sayHi() {
	try {
		const data = greeting();
		console.log('It worked!', data);
	} catch (e) {
		console.log('Oh no an error:', e);
	}
}

sayHi();
```
