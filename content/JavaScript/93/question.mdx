## 93. 以下代码会输出什么

```js:index.js
const handler = {
	set: () => console.log('Added a new property!'),
	get: () => console.log('Accessed a property!'),
};

const person = new Proxy({}, handler);

person.name = 'Lydia';
person.name;
```
