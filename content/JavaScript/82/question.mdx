## 82. 以下代码会输出什么

```js:index.js
function* generatorOne() {
	yield ['a', 'b', 'c'];
}

function* generatorTwo() {
	yield* ['a', 'b', 'c'];
}

const one = generatorOne();
const two = generatorTwo();

console.log(one.next().value);
console.log(two.next().value);
```
